Full Digital Marketing - High-Conversion Landing Page 
Bem-vindo ao repositório oficial da Full Digital Marketing. Este projeto é uma Landing Page de alta conversão, desenvolvida com tecnologias modernas para garantir performance, estética premium e resultados reais na captação de leads.

- Sobre o Projeto
Este projeto foi criado com o objetivo de ser a vitrine digital da nossa agência de marketing de performance. O foco é apresentar nossos serviços (Gestão de Tráfego, Landing Pages e Automação de Vendas) de forma persuasiva, com um design voltado para conversão e uma experiência de usuário fluida em qualquer dispositivo.

Principais Características:
Design Responsivo e Premium: Interface moderna com estética "dark mode" e acentos neon, otimizada para mobile e desktop.
Alta Performance: Carregamento ultra-rápido garantido pelo Vite e React.
Seções Estratégicas: Hero Section impactante, Prova Social (Depoimentos), Grid de Serviços Detalhado, FAQ Interativo e CTAs (Chamadas para Ação) otimizados.
Interatividade: Animações suaves, efeitos de hover e um botão flutuante do WhatsApp para contato imediato.
- Tecnologias Utilizadas
Este projeto foi construído utilizando as melhores ferramentas do mercado para desenvolvimento web:

React: Biblioteca JavaScript para construção da interface de usuário.
TypeScript: Superset do JavaScript que adiciona tipagem estática, garantindo um código mais seguro e escalável.
Vite: Build tool de próxima geração para um ambiente de desenvolvimento extremamente rápido.
Tailwind CSS: Framework de CSS utility-first para estilização rápida e responsiva.
Radix UI: Biblioteca de componentes acessíveis e sem estilo (headless) para a construção de interfaces robustas.
Lucide React: Coleção de ícones leves e consistentes.
- Como Executar o Projeto
Siga os passos abaixo para rodar o projeto em sua máquina local:

Clone o repositório:
bash
git clone https://github.com/Fehcoelhords/Full-digital-marketing--Leadpage.git
Acesse a pasta do projeto:
bash
cd "Full-digital-marketing--Leadpage"
Instale as dependências:
bash
npm install
Inicie o servidor de desenvolvimento:
bash
npm run dev
O projeto estará rodando em http://localhost:5173.
